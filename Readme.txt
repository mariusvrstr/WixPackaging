=============================================================
==================== .NET WIX Packaging =====================
=============================================================

Preparation:
1. Download and install WIX (https://wix.codeplex.com/releases/view/115492)
2. Create a basic MVC web project
3. Create a basic console application
4. Create WiX Setup Project for each of the projects (MVC and console apps)
5. Configure debug (configuration manage not to include WiX projects)
6. Add Nuget WiX Toolset (unofficial)
7. Download and Instlall WixEdit (http://wixedit.sourceforge.net)
   *This UI tool will help you to create the custom dialogs
